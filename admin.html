Adminstration System


Admin guide:

  • To create a NEW student,
    1. Fill in the form and
    2. Click 'Add New Student'
  • To EDIT/MAKE CHANGES to an existing student,
    1. Key in Student Name
    2. Click "Retrieve Data",
    3. Edit the relavant fields and then
    4. Click "Save"
  • To DELETE an existing student
    1. Key in Student Name
    2. click "Retrieve Data" (This is to check the correct student)
    3. Click "Delete"
  • To make *REMOVE ALL students
    1. click "Remove all records"







*Warning! This button removes all records!!!
